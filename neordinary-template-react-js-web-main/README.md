#

## Prerequisites
*
- npm = 8.5.0
- node = v16.14.2

#

## 설치법
*
```sh
npm install

or

npm install --force
```

#

## 로컬 실행법 / 배포 소스 빌드법
*
```sh
npm run start

npm run build-dev // Local 환경에서 개발을 완료하고 바로 서버로 Push 했을 경우 작동이 안되는 경우 (node version, os version, security group 보안설정)

npm run build-stage //

npm run build-prod //
```
